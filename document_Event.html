当前对象为：
Event
对象说明:
Event对象可以理解为静态对象，我们可以直接获取其内部方法。
如：Event.keyCode();或Event.fixEvent();
[注]在使用此对象时一定要在触发事件方法体内使用

方法详细信息:
fixEvent
fixEvent() 获取事件源（相当window.event）。
参数：
无。
返回：
事件源。 
例：Event.fixEvent();//[注]必须放在触发事件的方法里用

keyCode
keyCode() 获取 keycode 键盘码。
参数：
无。
返回：
键盘码。 
使用方法：
//页面加载完后对页面第一个输入框绑定键盘按下事件
$(function(){
	$("input").bind({
    	onkeydown:function(){
        	//提示按下的键盘码
			alert(Event.keyCode())
		}
    });	
});

stop
stop() 停止事件。
参数：
无。
返回：
无。 

element
element() 获取触发事件的对象。
参数：
无。
返回：
获取触发事件的对象。
