关于我们

感谢您访问JSEE（JavaScript Entity Engine 即 javascript 对象引擎）！

JSEE可以简化我们的 javascript 代码，内置多个java对象，我们可以像编写java代码那样来写您的javascript程序。 比如：new Map(); new Array(); new Thread(); new Timer()...等等;并且与java API基本吻合。强大的Jsee对象中提供了 丰富的常用方法与特效。JSEE兼容各种浏览器，各种终端，比如电脑、手机、机顶盒等。欢迎使用，真诚的期待与您交流！
