当前对象为：
PageContext
对象说明:
这个对象里存放了常用的方法和属性,使用时直接用对象名点方法或属性即可 如：PageContext.contextPath;//获取站点根路径
方法摘要:

返回值

方法名

方法说明

string

contextPath

获取站点根目录

string

param(name)

获取url中参数

string

encode(url)

将字符串转为URL编码

string

decode(url)

将URL编码的字符串解码

包含宽高对象

winSize()

获取窗口大小.

方法详细信息:
contextPath
contextPath 获取站点根目录
参数：
无。 
使用方法：PageContext.contextPath
返回：
站点根目录。 

param
param(name) 获取url中参数
参数：
name - 变量名。 
使用方法：PageContext.param(name);
返回：
url中变量。 

encode
encode(url) 将字符串转为URL编码
参数：
url - 要转码的字符串。 
使用方法：PageContext.encode(url)
返回：
转码后的字符串。 

decode
decode(url) 将URL编码的字符串解码
参数：
url - 要解码的字符串。  
使用方法：PageContext.decode(url)
返回：
解码后的字符串。 

winSize
winSize() 获取窗口大小
参数：
无。
返回：
包含宽高对象;
例：
PageContext.winSize().width;//获取窗口宽;
PageContext.winSize().height;//获取窗口高;
