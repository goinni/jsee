rtrim() 删除右边空格。

无

subString
subString(start,end) 截取字符串函数。
参数：
start - 开始位置
end - 结束位置
返回：
string。
说明
参数为负数时表示截取字符串到从后面数第几个，参数可以为一个也可为两个
比如："123456".subString(-2) 结果为1234
"123456".subString(1,-2) 结果为234

isBelong
isBelong(character)检查当前字符是否在给定的字符串内。
参数：
character - 为给定的字符,缺省值为"A~Za~z0~9"。
返回：
boolean。

toDate
toDate()将指定格式(yyyy-MM-dd hh:mm:ss)的日期字符串转为日期类型。
参数：
无。
可以转换的时间格式为以下几种类型
yyyy-MM
yyyy-MM-dd
yyyy-MM-dd hh
yyyy-MM-dd hh:mm 
yyyy-MM-dd hh:mm:ss
返回：
Date。 

byteSize
byteSize()获取字符串的字节长度。
参数：
无。
返回：
number。 
例如：
"1sS#符".byteSize(); 结果为6

cut
cut(len,f,p)分隔字符串。
参数：
len - 分隔长度
f - 1按字节分隔；0按字符串分隔。缺省值为0
p - 匹配分隔模式
返回：
数组。 
* 功能一: 按字符串长度分隔
* 功能二: 按字符串字节长度分隔
* 功能三: 可以指定分隔模式

isLeapYear
isLeapYear()判断年字符串是否为闰年。
参数：
无。
返回：
 如: '2016'.isLeapYear() = true;

check
check(d);求两个[指定格式](yyyy-MM-dd hh:mm:ss)日期字符大小,返回值为true,false,eq。
参数：
d-指定日期格式字符串。
返回：
 如: '2012-05-03'.check('2012-02-01') 结果为： true;
 '2012-05-03'.check('2012-06-01') 结果为： false;
 '2012-05-03'.check('2012-05-03') 结果为： eq;

distance
distance(d);求两个[指定格式]日期字符串的日期相隔长度，返回毫秒数。
参数：
d-指定日期格式字符串。
返回：
 如: '2012-05-03'.distance('2012-05-01')/(1*24*60*60*1000) 结果为2天(因为获取相隔 毫秒数 后又转成 天)

replaceAll
replaceAll(a,b);查找字符串中a的值用b来替换。
参数：
a-要查找替换的字符串。
b-用来取代a。
返回：
如：'我是沈阳人,我爱沈阳!'.replaceAll("沈阳","北京"); 结果为：我是北京人，我爱北京
