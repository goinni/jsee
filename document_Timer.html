Timer
对象说明:
Timer是一个简单的计时器，可以方便的获取执行任务的时间差，可以控制计时开始与停止

返回值

方法详细信息:
Timer
Timer(fn,time) 构造器
参数：
fn - 回调方法。[注]fn中的this指向当前timer对象; fn中可以接受一个参数 即：程序执行的时间差,即开始 至 现在 单位毫秒
time - 计时频率 [单位毫秒]
如：
new Timer(
	function(time){
    	//time可以省略
		if(time >= 6000) {
        	//this指向当前timer对象
			this.stop();
		}
		document.getELementById("div_id").innerHTML= "程序执行了： "+time+" 毫秒";
	},100).start();
