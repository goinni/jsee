format
format(pattern)格式化日期，返回日期字符串;
参数：
pattern - 指定日期格式(如：yyyy/MM/dd hh:mm:ss)[注]格式化的日期表示符必须与例子大小写相同
pattern 可以为各种格式
如：yyyy-MM-dd 或 MM/dd/yyyy 或yyyy/MM/dd hh:mm等
返回：
返回格式化后日期字符串
例：new Date().format("yyyy/MM/dd hh:mm");//格式化当前日期
new Date().format("yyyy年MM月dd日");

vary
vary(pattern)对日期年,月,日,时,分,秒,毫秒增减; 
参数：
pattern - 对日期操作的格式 ,支持 '+' or '-'
// 例如 '+'
//'+nY' //加n年
//'+nM' //加n月
//'+nD' //加n天
//'+nH' //加n小时
//'+nF' //加n分钟
//'+nS' //加n秒
//'+nE' //加n毫秒
// 返回Date类型
返回：
Date类型。 
例：new Date().vary("+2Y");对当前日期加两年
