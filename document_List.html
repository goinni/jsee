方法详细信息:
merge
merge(value) 合并两个数组。
参数：
value - 要添加到此列表中的元素的数组
返回：
无。 

add
add(value,index)将指定的元素插入此列表中的指定位置。index缺省值为列表中最后一位
参数：
value - 要插入的元素 
index - 指定元素所插入位置的索引,缺省值为列表中最后一位
返回：
无。 

remove
remove(index)移除此列表中指定位置上的元素,并返回被当前被删除的值
参数：
index - 要移除的元素的索引
返回：
返回被当前被删除的值。 

removeValue
removeValue(value)移除此列表中首次出现的指定元素，并返回被移除元素的索引（如果存在）
参数：
value - 要移除的元素的值
返回：
返回被移除元素的索引（如果存在）。 

set
set(index,value)用指定的元素替代此列表中指定位置上的元素
参数：
index - 要替代的元素的索引
value - 存储在指定位置上的元素 
返回：
无。 

get
get(index) 返回此列表中指定位置上的元素。
参数：
index - 要返回元素的索引。 
返回：
此列表中指定位置上的元素。 

size
size() 返回此列表中的元素数。
参数：
无。 
返回：
此列表中的元素数。 

contains
contains(entity) 如果此列表中包含指定的元素，则返回 当前元素索引,否则返回false。
参数：
entity - 测试此列表中是否存在的元素。 
返回：
如果此列表包含特定的元素，则返回 当前元素索引,否则返回false

clear
clear() 移除此列表中的所有元素。
参数：
无。 
返回：
无。

isEmpty
isEmpty() 如果此列表中没有元素，则返回 true。
参数：
无。 
返回：
如果此列表中没有元素，则返回 true。

unique
unique() 数组去重复,去掉数组中重复数据。
参数：
无。 
返回：
去重复后的新数组。
